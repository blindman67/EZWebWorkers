EZWebWorkers.js Image processing example.

Click process to process the image.Back to examples.

This example creates a web worker each time it is needed to process the image.

The filters are all convolution filters. The mix filter if selected applies both filters. The mix filter is applied and the brightness of the result is compared to the original pixel. The difference is used to mix the amount of the main filter per pixel.If the main filter is none then the filter result is the mix amount per pixel.
